# Iban API

**API Name:** `iban`
**Base URL:** `https://iban-api.sls.epilot.io`

API Backend for epilot Iban feature.

## Operations

| Operation | Method | Path | Summary |
| --------- | ------ | ---- | ------- |
| `validateIban` | POST | `/v1/public/iban:validate` | validateIban |

## Usage

```bash
epilot iban validateIban
```
